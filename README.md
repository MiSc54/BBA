# Bernds Body App v1.2.2

## Was ist neu

Im Reiter "Uebungen" gibt es in der Filter-Gruppe "Geraetetyp" jetzt
zusaetzlich die Option "Favoriten". Waehlst du sie aus, werden nur noch
Uebungen angezeigt, die mindestens eines deiner favorisierten Geraete
benoetigen.

## Update einspielen

1. GitHub-Repository -> "Add file" -> "Upload files"
2. Alle 8 Dateien aus diesem ZIP hochladen (ueberschreiben die alten)
3. "Commit changes", ca. 1 Minute warten
4. App neu laden (dank Auto-Update sollte kein manuelles Cache-Loeschen
   noetig sein)
